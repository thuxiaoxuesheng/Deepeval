# Support

DeepEye is shared as an active development preview. Public support is currently best-effort.

## Where to Ask

- Use GitHub issues for reproducible bugs and documentation problems.
- Use feature request issues for scoped product or architecture proposals.
- Do not post secrets, private data, database connection strings, tokens, or exploitable security details.

## Useful Bug Report Details

Please include:

- operating system
- Docker and Docker Compose versions
- Python, uv, Node.js, and npm versions if relevant
- exact command that failed
- logs or screenshots with secrets removed
- whether the issue occurs with a fresh `docker compose up --build`

## Security Issues

Follow [SECURITY.md](SECURITY.md). Do not disclose exploitable details publicly.
