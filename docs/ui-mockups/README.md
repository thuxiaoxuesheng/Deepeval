DeepEye UI SVG Mockups

Files
- `deepeye-workspace-main.svg`: main shell with sidebar, workspace surface, and assistant rail.
- `deepeye-add-datasource-modal.svg`: add data source modal with upload and database connection states.
- `deepeye-video-panel-progress.svg`: video panel progress state with playback surface header, progress bar, and preview boot cards.

Usage
- Open directly in a browser for review.
- Open in Adobe Illustrator and save as `.ai` if an AI deliverable is required.
- All text, cards, icons, and layout blocks remain editable SVG vectors.

Design direction
- Based on the repo's current light workspace shell.
- Uses the existing accent family around `#49b6a6`.
- Keeps the product information architecture: sidebar, workspace tabs, assistant rail, and attached data management.
